set_pattern(pattern: list[tuple[int, int, int]], wraparound: bool = True, reset_time: bool = True) → None[source]

Sets a new pattern. Patterns are defined as a list of tuples (duration_ms, intensity1, intensity2). Each tuple represents a segment with a duration in milliseconds. Example: [(100, 5, 5), (200, 0, 0)] sets intensity1=intensity2=5 for 100ms, then intensity1=intensity2=0 for 200ms. An empty list means that no pattern is set and allows the toy to be controlled manually.

Args:

pattern: Pattern to be set wraparound: Whether the pattern should wrap around and be repeated indefinitely reset_time: Whether to reset the pattern playback position.
